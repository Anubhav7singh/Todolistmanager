# To-Do List Manager

A simple and efficient To-Do List Manager web application to help you keep track of your tasks and stay organized.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- User-friendly interface

## Technologies Used

- HTML
- CSS Tailwind
- JavaScript
- React.js
